## 🚀 Projects

**[World-Models-Explorer](https://github.com/ryadg-kura/world-models-explorer)** &nbsp;`Python` `PyTorch` `Gymnasium`
*World Models · Model-Based RL*
From-scratch RSSM implementation. Learns compact latent representations of environment dynamics and generates imagined future trajectories without interacting with the real world.

**[MiniRAG](https://github.com/ryadg-kura/minirag)** &nbsp;`FAISS` `sentence-transformers` `Ollama`
Minimal RAG pipeline from scratch, no LangChain, no abstraction layers. FAISS vector store, local embeddings, pluggable LLM backend.

**[SentimentCam](https://github.com/ryadg-kura/sentimentcam)** &nbsp;`OpenCV` `HuggingFace` `MediaPipe`
Real-time emotion analysis from live video. Face detection pipeline feeds into a transformer-based classifier, with emotion labels and confidence overlaid on the frame.

**[TextStyleTransfer](https://github.com/ryadg-kura/textstyletransfer)** &nbsp;`PyTorch` `HuggingFace` `Gradio`
Fine-tuned GPT-2 for formal to informal style transfer. Full training pipeline, BLEU/BERTScore evaluation, interactive Gradio demo.
